# MÓDULO EXTRA: Monitoramento de UPS, que de acordo com o google UPS significa "Uninterruptible Power Supply", ou fonte de alimentação ininterrupta, que é um dispositivo que fornece energia de emergência para equipamentos eletrônicos quando a energia da rede falha. Além de fornecer energia de backup, um UPS também pode proteger equipamentos contra picos de tensão, surtos e outras variações.
